Evaporative Cooler AC with ANN Control

Hybrid Cooling System Optimized with Artificial Neural Network

Project Overview

This project explores a hybrid air-conditioning system that combines evaporative cooling with traditional methods, managed using an Artificial Neural Network (ANN) controller for optimal performance and energy savings.

Key Features

  • Integrated ANN to predict and control temperature and humidity levels.
  • Hybrid cycle to minimize electrical consumption and improve efficiency.
  • Experimental validation and comparison with standard AC systems.
