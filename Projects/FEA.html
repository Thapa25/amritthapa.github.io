Nitsche's Method in FEA

Handling Multi-Patch Geometries with Non-Conforming Meshes

Project Overview

This project explores the application of Nitsche's Method in Finite Element Analysis (FEA) to address challenges in multi-patch geometries with non-conforming meshes. Nitsche's formulation adds interface terms to enforce continuity and boundary conditions while preserving stability and accuracy across disconnected mesh patches.

It excels in linear elasticity and complex domains, using jump and average operators to bridge mesh gaps, delivering high-fidelity solutions where standard FEA struggles.
